What is Leaves Life?

Welcome to our machine learning project. We use different supervised learning techniques in order to detect diseases in leaf pictures. Learn more about our project here.

How to use it?

You only have to insert a leaf image that is supported by our project, and it will detect any disease or the healthyness of the leaf!
